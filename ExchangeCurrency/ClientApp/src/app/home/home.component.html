Ejercicio - Entrevista - Dirmod v1.0

Desarrollado por: Gustavo Jimenez, usando:

  • ASP.NET Core y C# para codificación cross-platform server-side
  • Angular y TypeScript para código del client-side
  • Bootstrap para formato y estilo

Requerimientos:

  • Implementar una Web Api RESTful que retorne las cotizaciones de, al menos, 3 monedas.
  • Obtener Cotizacion Tasa de Cambio de: BCRA ó api.cambio ó api.wahrungsrechner.org ó apilayer.net/api/
  • Implementación de FrontEnd/Ux/Interfaz de Usuario para visualizar las 3 cotizaciones, refrescandose automáticamente cada 5 seg

Para acceder a las API de cotizaciones de tasa de cambio es requerido una Clave de acceso. Normalmente hay un límite de las consultas para las cuentas gratis. Si se hacen consultas cada 5 seg, el sistema podria alcanzar el máximo muy rapido arrojando un error.
